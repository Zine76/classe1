# Assets — Avancement à la classe 1

Ce dossier contient les ressources visuelles pour le site de présentation du dossier d'avancement.

## Structure

```
assets/
├── README.md           # Ce fichier
├── favicon.ico         # Favicon du site (optionnel)
├── stockav/            # Images du projet StockAV
│   ├── stockav-01.png  # Interface principale
│   ├── stockav-02.png  # Recherche IA
│   └── stockav-03.png  # Architecture système
└── vitrine/            # Images du projet Vitrine
    ├── vitrine-01.png  # Interface usager
    ├── vitrine-02.png  # Mode technicien
    └── vitrine-03.png  # Widget monitoring
```

## Spécifications des images

### Format recommandé
- **Format** : PNG ou JPEG
- **Ratio** : 16:10 (ex: 1600×1000 pixels)
- **Taille max** : 500 KB par image
- **Résolution** : 72-150 DPI pour le web

### Images StockAV

| Fichier | Description | Contenu suggéré |
|---------|-------------|-----------------|
| `stockav-01.png` | Interface principale | Capture de l'interface de gestion d'inventaire avec la liste des composants |
| `stockav-02.png` | Recherche IA | Capture de la fonction de recherche intelligente avec résultats |
| `stockav-03.png` | Architecture | Diagramme d'architecture technique (Mermaid ou draw.io) |

### Images Vitrine

| Fichier | Description | Contenu suggéré |
|---------|-------------|-----------------|
| `vitrine-01.png` | Interface usager | Capture de l'interface publique de signalement |
| `vitrine-02.png` | Mode technicien | Capture du mode technicien avec plans PDF |

## Notes

- Si les images ne sont pas présentes, le site affiche des placeholders élégants
- Les images sont chargées de manière asynchrone pour optimiser les performances
- Le site reste fonctionnel et professionnel même sans images

## Création des images

Pour créer des captures d'écran de qualité :

1. **Captures d'écran** : Utilisez l'outil de capture Windows (Win+Shift+S) ou un outil dédié
2. **Diagrammes** : Utilisez Mermaid, draw.io, ou Excalidraw
3. **Optimisation** : Compressez les images avec TinyPNG ou Squoosh
